## Kurzbeschrieb

Booze & Breeze schlägt dir passend zum aktuellen Wetter das perfekte Getränk vor. Die Webseite checkt die Bedingungen an der Aare (von Brienz bis Brugg) und liefert dir direkt die passende Cocktail-Empfehlung dazu.

## Learnings

API-Einbindung: Wir haben gelernt, wie man Daten von externen Webseiten (Wetter und Cocktail-Datenbanken) anzapft und auf der eigenen Seite anzeigt.
GitHub: Wir wissen jetzt, wie man Code im Team via GitHub teilt, ohne sich gegenseitig alles zu überschreiben.
Coding-Skills: Unsere Programmierkenntnisse (vor allem in JavaScript und CSS) haben durch das Projekt sich klar verbessert.

## Schwierigkeiten

Responsive Design: Es war echt knifflig, die Seite so zu bauen, dass sie auf dem Handy genauso gut aussieht und funktioniert wie auf dem Laptop.
Interaktive Effekte: Manche Funktionen, wie zum Beispiel die Hover-Effekte, haben auf verschiedenen Geräten nicht auf Anhieb so reagiert, wie wir uns das vorgestellt hatten.

## Know Bugs

Hängende Texte: Manchmal wechselt der Text beim Getränk nicht richtig. Wenn man mehrmals klickt, "klemmt" es ab und zu und es wird derselbe Text noch einmal angezeigt.

## Ressourcen

Wetterdaten: [Aareguru Widget API](https://aareguru.existenz.ch/v2018/widget)
Cocktail-Rezepte: [TheCocktailDB API](https://aareguru.existenz.ch/v2018/widget)

## Sonstiges

Die Sache mit der Temperatur: Uns ist erst ganz am Schluss aufgefallen, dass die API von Aareguru uns die Wassertemperatur der Aare liefert und nicht die Lufttemperatur. Für ein Projekt am Fluss passt das aber eigentlich sogar ziemlich gut!
